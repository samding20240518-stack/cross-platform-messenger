## 🚀 上传步骤

### 步骤1: 检查远程仓库配置
```bash
# 检查当前远程仓库
$ git remote -v

# 如果还没有配置远程仓库，添加它（替换为你的GitHub用户名）
$ git remote add origin https://github.com/你的用户名/cross-platform-messenger.git
```

### 步骤2: 验证当前分支
```bash
# 检查当前分支
$ git branch

# 确保在main分支（或master分支）
$ git checkout main  # 或 master
```

### 步骤3: 推送新版本到GitHub
```bash
# 推送v0.2.0版本到GitHub
$ git push origin main

# 如果这是首次推送，可能需要设置上游分支
$ git push -u origin main
```

### 步骤4: 创建GitHub Release（推荐）

在GitHub网站上：
1. 进入你的仓库页面
2. 点击 "Releases" → "Draft a new release"
3. 输入标签名: `v0.2.0`
4. 输入发布标题: `v0.2.0 - 完整音效系统`
5. 输入发布说明（复制下面的内容）:
